Redactor Plugins

文本扩展

输入文字或文字的简短片段，这个插件将它替换到一个经常使用的预定义的文本。例如，输入“加法”，以让它在您的邮寄地址替换.

Download Plugin 

<!DOCTYPE html>
<html>
    <head>
        <title>Redactor</title>
        <meta charset="utf-8">

        <script src="//cdnjs.cloudflare.com/ajax/libs/jquery/2.1.1/jquery.js"></script>

        <link rel="stylesheet" href="../../js/redactor/redactor.css" />
        <script src="../../js/redactor/redactor.js"></script>

        <!-- Plugin -->
        <script src="textexpander.js"></script>

        <script type="text/javascript">
        $(function()
        {
            $('#redactor').redactor({
                plugins: ['textexpander'],
                textexpander: [
                    ['lorem', 'Lorem ipsum...'],
                    ['text', 'Text']
                ]
            });
        });
        </script>
    </head>
    <body>
        <textarea id="redactor">...</textarea>
    </body>
</html>
